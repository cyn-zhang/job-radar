# source-command-job-cv

Use this skill when the user asks to run the migrated source command `job-cv`.

## Command Template

Tailor my CV for a specific role using JobRadar. Load config from config.yaml, then follow Module 3 in SKILL.md. Load the CV from the configured cv_path, cherry-pick the strongest evidence for the JD, apply adjacent framing where needed, and generate a tailored CV. Never fabricate or pad experience.

If no role/company context is provided, ask: "Which role and company should I tailor your CV for?"
