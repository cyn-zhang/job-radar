# source-command-job-prep

Use this skill when the user asks to run the migrated source command `job-prep`.

## Command Template

Run interview prep using JobRadar. Load config from config.yaml, then follow Module 6 in SKILL.md. Generate 8 behavioural questions (STAR format) and 10 technical questions tailored to the role. Offer a mock interview mode.

If no role/company context is provided, ask: "Which role and company are you prepping for?"
