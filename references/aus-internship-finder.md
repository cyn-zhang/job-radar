## How to Use in Job Scan

**Do NOT search `site:aus-internship-finder.vercel.app`** — it's a single-page app, not indexed.

Instead: use the directory below to know which companies to check directly, and what timing to expect.

```
# Search pattern for each company in the finder:
"{company_name}" careers internship {year}
"{company_name}" graduate program application
site:seek.com.au "{company_name}" intern
```

## Critical Timing Summary

```
Jan–April   ⚠️  Quant Trading apps open & close (Optiver, Citadel, IMC, SIG)
Feb–March   ⚠️  MBB consulting closes (McKinsey, Bain)
Feb–April   ⚠️  Atlassian closes
April–May   ⚠️  Canva closes
June        →   AustralianSuper opens
July–Aug    →   Main wave: Big 4, Banks, IBM, Oracle, Telstra, Xero, Microsoft
Aug–Sept    →   Amazon, Microsoft, Aware Super, L.E.K., Bank of America
Rolling     →   Google (from Feb), Amazon (July–Sept), TikTok (Jan–Mar)
Nov–Feb     →   Programs run (Summer internship period)
```

**Rule of thumb:** If you want Quant, MBB, Atlassian, or Canva — apply 6–9 months before the program starts.
For everyone else — apply July/August before a November start.

## Visa Notes

- **"Full working rights required"** = Australian citizen or permanent resident only (no international student visa)
- **"International OK (case-by-case)"** = international students on student visas may be considered; apply and declare your visa status
- When in doubt, check the company's career page directly — visa policies change year to year
